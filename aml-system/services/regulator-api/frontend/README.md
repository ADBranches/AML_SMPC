# AML SMPC Regulator Frontend

## Overview

Browser-based regulator console for the AML SMPC prototype.

## Stack

React + Vite + TypeScript + Tailwind CSS

## Environment

cp .env.example .env

## Install

npm install

## Run

npm run dev -- --host 127.0.0.1 --port 5173

## Build

npm run build

## Demo

From project root:

./scripts/demo/run-frontend-demo.sh

## Main Routes

- /dashboard
- /proofs
- /audit
- /performance
- /about
